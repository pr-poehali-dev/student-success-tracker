# Шаблоны импорта данных

## 1. Импорт классов и учеников

### Формат файла Excel:

Файл должен содержать 2 листа:

#### **Лист "Классы"**
| Название |
|----------|
| 5-А      |
| 5-Б      |
| 6-А      |

#### **Лист "Ученики"**
| ФИО              | Класс | Баллы |
|------------------|-------|-------|
| Иванов Иван      | 5-А   | 100   |
| Петрова Мария    | 5-А   | 150   |
| Сидоров Петр     | 5-Б   | 80    |

**Примечания:**
- Столбец "Баллы" необязателен (по умолчанию 0)
- Если класс не указан в листе "Классы", он будет создан автоматически
- Дубликаты учеников и классов не добавляются повторно

### Как импортировать:
1. Перейдите во вкладку **"Классы"**
2. Нажмите кнопку **"Импорт из Excel"**
3. Выберите файл с нужной структурой
4. Данные загрузятся в систему

---

## 2. Импорт команд для матчей

### Формат файла Excel:

Файл должен содержать 1 лист:

#### **Лист "Матчи"**
| Тип игры      | Команда 1  | Команда 2   | Участники команды 1                | Участники команды 2                |
|---------------|------------|-------------|------------------------------------|------------------------------------|
| valheim       | Драконы    | Викинги     | Иванов Иван, Петрова Мария         | Сидоров Петр, Кузнецова Анна       |
| sport         | Команда А  | Команда Б   | Смирнов Алексей, Васильев Дмитрий  | Николаев Сергей, Федоров Максим    |
| civilization  | Империя    | Республика  | Попов Андрей, Морозова Елена       | Соколов Виктор, Павлова Ольга      |

**Доступные типы игр:**
- `valheim` - Вальхейм
- `civilization` - Цивилизация
- `factorio` - Факторио
- `sport` - Спорт
- `robo` - Робо

**Примечания:**
- Участники указываются через запятую
- ФИО участников должны точно совпадать с учениками в системе
- Если ученик не найден, он будет пропущен
- Все участники автоматически получают роль "Игрок"
- После импорта нужно нажать кнопку **"Создать матч"**

### Как импортировать:
1. Перейдите во вкладку **"Команды"**
2. Нажмите кнопку **"Импорт команд"**
3. Выберите файл с матчем
4. Команды загрузятся в форму создания матча
5. Нажмите **"Создать матч"** для сохранения

## 3. Экспорт результатов

### Что содержится в экспорте:

Файл Excel с несколькими листами:

#### **Лист "Общая сводка"**
- ФИО ученика
- Класс
- Люмосити (баллы)
- Робо (время мин)
- Спорт Побед
- Спорт Проигрышей
- Спорт Капитаном
- Спорт Игроком

#### **Листы для каждой игры** (Спорт, Вальхейм, Цивилизация, Факторио, Робо, Люмосити):
- ФИО
- Класс
- Дата
- **Название матча** (например: "Драконы vs Викинги")
- **Команда** (название команды ученика)
- **Противник** (команда соперников)
- Роль (Капитан / Игрок)
- Результат (Победа / Поражение)
- Статус игры (Закончена / Идет игра)

### Как экспортировать:
1. Перейдите во вкладку **"Экспорт"**
2. Нажмите **"Скачать общий Excel"** для всех классов
3. Или выберите конкретный класс и нажмите **"Экспорт"**
4. Файл автоматически скачается на ваш компьютер

## Примеры использования

### Пример 1: Начало учебного года
1. Создайте Excel с классами и учениками
2. Импортируйте через вкладку "Классы"
3. Начните добавлять активности

### Пример 2: Проведение турнира
1. Создайте Excel с матчами и составами команд
2. Импортируйте через вкладку "Команды"
3. Создайте матчи и назначьте победителей
4. Экспортируйте результаты для отчёта

### Пример 3: Анализ результатов
1. Экспортируйте данные в Excel
2. Откройте файл и просмотрите листы
3. Используйте фильтры Excel для анализа:
   - Кто играл в каких матчах
   - Какие команды выигрывали
   - Статистика по каждому ученику
